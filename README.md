# 🍛 THE RAHA - Cloud Restaurant POS System

Complete cloud-based Point of Sale system for Irish restaurants.

## Features
- ✅ Real-time order sync (counter → kitchen < 1 second)
- ✅ Kitchen Display System with live updates
- ✅ Admin monitoring from anywhere
- ✅ 72-item menu pre-loaded
- ✅ Dynamic VAT rates (13.5% → 9% from July 2026)
- ✅ Irish compliance ready
- ✅ Admin delete powers
- ✅ Offline queue support

## Quick Start

### Local Development
```bash
npm install
node server.js
```
Open http://localhost:3000

### Deploy to Railway.app
1. Push to GitHub
2. Connect Railway to repo
3. Auto-deploys
4. Get your URL!
